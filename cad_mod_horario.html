COMO CRIAR UM MODELO DE HORÁRIO

*****


1º) No exemplo abaixo criarei um modelo de horário para uma turma do ensino fundamental II manhã, lembre-se de criar para todos os níveis (séries) e turnos (manhã, tarde, etc)

Clicar no menu Horário > Modelo de horário > Adicionar;


2º) Preencher as informações: obs., procure sempre manter um padrão para o código;


3º) Ao clicar no botão Adicionar tempo, vamos criando as aulas conforme o tempo de cada uma delas;


4º) Se clicarmos no botão Copiar para, o modelo é criado semelhante para o próximo dia ou dia escolhido;


5º) Veja:


6º) Para ativar o modelo de horário clique em Editar:


7º) Clique no botão Ativo e depois em Salvar:
